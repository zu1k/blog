学校又开始推雨课堂的在线考试功能了，在老师的配合下稍微分析了一下

因为都是js代码，没有用wasm，分析还是比较简单的

切屏检测

考试过程中，如果切换浏览器标签，返回后会有一个切屏警告，并且老师那边也会有切屏记录

通过实验发现如果不切换浏览器标签，使用另一个浏览器窗口或者切换其他软件都不会触发这个记录
