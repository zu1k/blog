е®үиЈ…Wrangler

е®ҳж–№ж–ҮжЎЈ

WranglerжңүдёӨз§Қе®үиЈ…ж–№ејҸпјҢйҖҡиҝҮNPMжҲ–иҖ…Cargoе®үиЈ…йғҪеҸҜд»ҘпјҢд»»йҖүе…¶дёҖеҚіеҸҜ

еҮҶеӨҮеҘҪNodeJSе’ҢNPMзҺҜеўғпјҢз„¶еҗҺжү§иЎҢдёӢйқўе‘Ҫд»ӨпјҢNPMж–№ејҸжҳҜдёӢиҪҪйў„зј–иҜ‘еҘҪзҡ„дәҢиҝӣеҲ¶зЁӢеәҸпјҢе®үиЈ…йҖҹеәҰжҜ”иҫғеҝ«

account_idе’Ңzone_idйғҪеҸҜд»Ҙд»ҺCloudflareе®ҳзҪ‘дёҠжүҫеҲ°пјҢrouteжҳҜи·Ҝз”ұеҲ°Workersзҡ„и§„еҲҷпјҢиҝҷйҮҢеҶҷдҪ йңҖиҰҒз»‘е®ҡзҡ„еҹҹеҗҚпјҢдёҚиҰҒеҝҳи®°еҗҺйқўзҡ„/*

bucketжҳҜзҪ‘з«ҷзҡ„зӣ®еҪ•пјҢеӣ дёәжҲ‘з”Ёзҡ„жҳҜHugoпјҢжүҖд»ҘиҝҷдёӘзӣ®еҪ•й»ҳи®ӨжҳҜpublic

name: hugo

on:
  push:
    branches:
      - master

jobs:
  deploy:
    runs-on: ubuntu-18.04
    steps:
      - uses: actions/checkout@v2
        with:
          submodules: true
          fetch-depth: 0

      - name: Setup Hugo
        uses: peaceiris/actions-hugo@v2
        with:
          hugo-version: '0.78.2'
          extended: true

      - name: Build
        run: hugo --minify

      - name: Deploy
        uses: peaceiris/actions-gh-pages@v3
        with:
          github_token: ${{ secrets.GITHUB_TOKEN }}
          publish_dir: ./public

      - name: Publish
        uses: cloudflare/wrangler-action@1.3.0
        with:
          apiToken: ${{ secrets.CF_API_TOKEN }}
